# robolab renderer

Rewrite of the robolab rendering egnine. 

## Features

- [x] Visualise planets
- [x] Planet editor
- [x] Animation support **new**
- [x] Multiplatform support (ui abstraction) **new** 

## Usage

```bash
./gradlew run
```
or
```
./gradlew jvmJar
java -jar build/libs/robolab-jvm.jar
```
